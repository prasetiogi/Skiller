All notable changes to this skill will be documented in this file.

## [1.0.0] - 19 Feb 2026 01:49

### Added

- Initial release of pdf-editor skill
- `scripts/rotate_pdf.py` — rotate all or specific pages by 90°, 180°, or 270° (clockwise)
- `scripts/merge_pdf.py` — merge two or more PDF files in specified order
- `scripts/split_pdf.py` — split PDF into individual pages or extract specific page ranges
- `scripts/compress_pdf.py` — compress PDF via stream compression and object deduplication
- `references/pdf-libraries.md` — library selection guide (pypdf, pdfplumber, pikepdf, Ghostscript), API quick reference, and troubleshooting table
- SKILL.md with Task-Based structure pattern and journey description formula
